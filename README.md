# 뭉게뭉게 (MGMG)
![](./Manual/Pictures/readme_img4.jpeg)
 - CasperJS를 이용한 서버 모니터링 모바일 앱 & 구름조명 알람
 - [MGMG 소개 홈페이지](http://1.234.20.120)



## 개요
 - 웹 서비스의 Health 상태를 CaperJS를 이용하여 모니터링 하고, 결과를 모바일과 Raspberry Pi 기반의 Off-Line LED 구름으로 표시 한다.
